# Fintech-Customer-Churn-Project

## Project overview

This project analyses customer transaction behaviour for a fictional fintech company.

The objective is to identify customers who may become inactive, understand the behaviours associated with churn, and present actionable insights through an interactive dashboard.

## Business problem

Customer churn can reduce revenue and increase customer acquisition costs. Predicting churn early can help a fintech company take targeted action to retain valuable customers.

## Initial project objectives

- Explore customer and transaction behaviour
- Define a measurable churn outcome
- Engineer meaningful behavioural features
- Build and evaluate a churn prediction model
- Explain individual customer predictions
- Develop an interactive business dashboard
- Add an AI-powered natural-language analysis feature

## Tools

Python, SQL, Pandas, Scikit-learn, Streamlit, Plotly, Git and GitHub.
